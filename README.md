# ATUA-ASSIGNMENT2-HH

## PM2.5 Prediction Using Graph Neural Networks

## Overview

This project explores the use of Graph Neural Networks (GNNs) to predict PM2.5 air pollution levels using spatial relationships between monitoring stations. Two models are implemented and compared:

## Graph Convolutional Network (GCN) & GraphSAGE

The objective is to evaluate how effectively graph-based models capture spatial dependencies in air pollution data and improve prediction performance.

The results show a positive relationship between predicted and actual PM2.5 values, with GraphSAGE performing slightly better, particularly for extreme pollution levels.
